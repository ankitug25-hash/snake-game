
# 🐍 Snake Game in C++

A classic, console-based Snake Game built using C++. This project demonstrates foundational programming concepts such as Object-Oriented Logic, Vectors, and Windows Console Management.

## 🎮 How to Play
Use the following keys on your keyboard to control the snake:
* **W** - Move Up
* **S** - Move Down
* **A** - Move Left
* **D** - Move Right
* **X** - Exit Game

## 🛠️ Features Implemented
* Dynamic snake body growth using `std::vector`.
* Smooth screen transitions with minimized flickering using Windows Console API (`SetConsoleCursorPosition`).
* Score tracking system.
* Self-collision and boundary-collision detection.

## 💻 Tech Stack
* **Language:** C++
* **Libraries:** `<iostream>`, `<vector>`, `<conio.h>`, `<windows.h>`
